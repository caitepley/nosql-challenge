# nosql-challenge
The purpose of this assignment is to update and analyze restaurant data contained in a MongoDB database. The file called 'NoSQL_setup.ipynb' contains a Jupyter notebook that is used to update the contents of the database so that only useful information is left and it is in a form that is usable.
